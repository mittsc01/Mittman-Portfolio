Travelling salesman

The Travelling Salesman app allows the user to create an account and then create routes using an interactive map. The user is then able to add routes to their schedule. I created this to teach myself how to make an app with React Leaflet that allows users to create routes.

Technologies

Race Director

The Race director app allows the user to create an account and start creating races with basic information (date, start time, distance, city, state, etc). Users can add manage results for the races they create as well as view other users' races. I built this because I wanted to provide an easy way to share races and race results.
